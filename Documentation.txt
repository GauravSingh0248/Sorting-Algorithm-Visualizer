📌 Overview
Sorting Algorithm Visualizer is a web-based tool that demonstrates the inner workings of 7 sorting algorithms through real-time animations and performance comparisons. This interactive tool is built for students and developers to learn, compare, and understand sorting visually.

✨ Features

Bubble, Selection, Insertion, Merge, Quick, Heap, and Radix Sort

🧠 Shows:

Time complexity

Number of comparisons

Time taken

🖊️ Custom input and random array generator

⚙️ How It Works
Select a sorting algorithm from the dropdown.

Choose input method: random, custom, ascending, descending.

Adjust array size and speed with sliders.

Click Start Sorting or Step Mode to visualize.

View metrics like comparisons, time taken, and complexity.
